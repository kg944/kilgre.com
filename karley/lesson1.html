First assignment

My laptop battery is dying so I'm going to write some short things down here but clean it up and expand upon it later. As a refresher, our next project is to make a guessing game! In a sentence, this program will generate a random number, and accept guesses from the user until they have guessed the correct number, telling the user if their guess is too high or too low after every guess. To break it down more into more easily programmable steps:

  1. Generate a random number
  2. Ask for the user's first guess
  3. While the user has not yet guessed our number, ask the user for a new guess.
  4. If this is the correct number, print success and stop the program!
  5. Otherwise if their guess is too high print "too high" or if too low print "too low".
  6. Repeat from step 3 until success!

Input

Accepting user input can be done with the input() function. We can set the return of this function equal to a variable in order to store the user input for later use. An example:

user_name = input("enter your name: ")

The above code will prompt the user for their name, and store it to variable user_name

For loops

For loops are useful to do something a set number of times, when we know ahead of times exactly how many iterations we want. They are also useful for looping over arrays, and when we want to know what iteration of the loop we are on (think printing something like "item 1: blah bla", "item 2: blah blah" where 1 and 2 are 'i' in our loop. Example of a simple for loop:

        for i in range(0, 5):
            print(i)

While loops

While loops are useful for continuing to loop while a condition is still true. Reminder: a condition is a true or false statement, for example x < 10 or user_input != "quit". An example of a simple while loop:

      while count < 10:
          print("count is less than 10!")
          count += 1

If statements

If statements are used to test a condition, and perform some logic if it is true. You can add "else if"s and "else"s after the if statements to tell the computer to check another condition if the first is false, or to do something if the above statement is false respectively. That was probably a bad way to word it but you're smart. Example of if statements:

        if age >= 21:
            print("you can drink")
        else if age >= 18:
            print("you can vote")
        else if age == 15:
            print("you are exactly 15 years old")
        else:
            print("ur a lil baby wah wah also you're not 15")
